# Pengenalan ke Obserfy

Selamat datang di Obserfy! Halaman ini dapat membantu anda untuk mempelajari lebih lanjut tentang fitur-fitur Obserfy
dan cara untuk membuat Obserfy seberguna mungkin untuk anda.

## Fitur Utama

Obserfy adalah software untuk membantu sekolah Montessori mengurus data murid dan berkomunikasi dengan orang tua.

Kami membuat Obserfy untuk membantu meringankan pekerjaan guru dan untuk menjembatani komunikasi antar sekolah dan wali
murid/orang tua agar lebih efisien.

Berikut adalah 2 dashboard yang kami sediakan untuk anda dan fitur-fitur utama dari masing-masing dashboard tersebut.

## Obserfy untuk Guru

Dashboard guru ber-fokus untuk menaikan produktifitas guru. Guru dapat mengatur semua data sekolah anda yang anda simpan
dalam Obserfy melalui dashboard ini.

Anda dapat mengakses dashboard ini melalui https://app.obserfy.com. Di dashboard ini guru bisa:

- Mencatat observasi tentang murid
- Merencanakan pembelajaran murid yang akan datang
- Mencatat progres murid dalam kurikulum sekolah anda
- Menyimpan foto dari aktifitas murid di sekolah


## Obserfy untuk Wali Murid

Dashboard wali murid berfokus untuk memberikan cara yang mudah dan cepat untuk wali murid/orang tua untuk mengakses data
tentang anak mereka.

Orang tua dapat menggunakan dashboard ini untuk mendapatkan panduan untuk menjalankan pembelajaran di rumah dan untuk
mengikuti info kegiatan anak mereka di sekolah.

Ini berguna terutama se-masa pandemi, dimana anak-anak terpaksa harus diam dirumah

Anda dapat mengakses dashboard ini dengan mendatangi https://parent.obserfy.com. Disini orang tua/wali murid dapat:

- Melihat observasi
- Melihat rencana pembelajaran
- Memasang observasi terkait ke rencana pembelajaran di rumah
- Melihat foto yang dipasangkan oleh guru
- Meng-upload foto yang mereka tangkap sendiri
