# Mendapatkan Bantuan

Kita akan dengan senang hati mendengar kebutuhan guru dan orang tua/wali murid :). Ada beberapa cara untuk menghubungi kami.

## Canny

Anda bisa memberi kritik/saran melalui Canny. Semua kritik/saran disini akan dapat dilihat semua orang. Semua post yang
dibuat disini akan kami gunakan secara langsung untuk menyusun roadmap fitur Obserfy kedepan.

Anda bisa membuka Canny dengan mengunjungi [feedback.obserfy.com](https://feedback.obserfy.com). Atau dengan meng-klik link
di halaman support/bantuan dari kedua dashboard untuk Guru dan untuk wali wurid

Anda mungkin akan memerlukan akun baru khusus untuk Canny sebelum anda dapat membuat postingan baru disini.

## Live Chat

Kami menyediakan live chat di website kami, anda dapat menemukan ini di kanan bawah layar anda di desktop dan tablet.
Live chat akan muncul sekitar 3-5 detik setelah website selesai loading.

Untuk sekarang live chat tidak tersedia di smartphone karena ini merupakan fitur yang lumayan berat untuk dijalankan.

## Email

Anda juga bisa menghubungi kami melalui [help@obserfy.com](mailto:help@obserfy.com)
You can also contact us by email through help@obserfy.com
