# Format teks menggunakan Markdown

Markdown adalah sebuah cara untuk mem-format teks menggunakan symbol. Anda dapat dengan mudah menambahkan link, menambah judul,
dan memngubah teks anda menjadi _bold_ atau _italic_ menggunakan markdown.

## Cara penggunaan

Di Obserfy, anda bisa menggunakan markdown di tempat yang memiliki tulisan "Markdown Tersedia".

![Markdown supported icon](../../images/markdown-illustration.png)

Anda bisa menulis teks anda di tab "Write". Dan anda bisa melihat hasil foratannya pada tab "Preview".

## Format yang tersedia

---

### Italics

#### <u>Syntax</u>

```markdown
_text_
```

#### <u>Hasil</u>

_text_

---

### Bold

#### <u>Syntax</u>

```markdown
**text**
```

#### <u>Hasil</u>

**text**

---

### Links

#### <u>Syntax</u>

```markdown
[this is a link](https://example.com)
```

#### <u>Hasil</u>

<div />

[this is a link](https://example.com)

---

### Headings

#### <u>Syntax</u>

```markdown
# Heading 1

## Heading 2

### Heading 3

#### Heading 4
```

#### <u>Hasil</u>

<h1 className={"my-2"}>Heading 1</h1>
<h2 className={"my-2"}>Heading 2</h2>
<h3 className={"my-2"}>Heading 3</h3>
<h4 className={"mb-8"}>Heading 4</h4>
